Rename deformer nodes

Tool to rename all deformer nodes based on node type.

-Node type list
    Loads all deformer nodes on selected object when the tool opens.
    if no object is selected loads empty.
    select object and click Refresh node list to siaplay deformer nodes on object.

-Replace text/ Replace with
    -All nodes of selected type gets renamed.
    -reads the object name and replaces the string part mentioned in
    replace text with text in replace to name the deformer node.

-Rename
    -Renames the nodes of selected node type, if more than one node of same
    type exists, number is added to the end.
