Immediate parent in hierarchy
    create a parent node for selected object with povit at selected object pivot

hide joint/show joint
    turn on or off selected joint visibility

LOD off/LOD on
    turn on or off LOD of objects(Works for objects with LOD property)

Parent in selection order
    parent nodes in selected order with the first selection being on top

Copy objects to selected positions
    Copies the assigned object at points selected(object or components)
    check parent or/and scale to create constraints with the object to act as controls

Copy orientation
    Copy joint orientation from forst selected joint to the rest of selection
