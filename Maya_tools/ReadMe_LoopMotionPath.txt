Tool to create motion path looping in closed curve
enter name for the setup
input curve

uniform:
    palces objects/joints at uniform intervals based on number specified

selection:
    -place objects or reference along curve, select objects or references,
    creates motionpath objects on the curve closest to the objects selected
    -Can either use the selected objects themselves or can use a reference
    object to place along curve at selected positions

input sample object:
    assign object that will be placed along curve (default joint).
